Getting Started

  1. Launch FDUtility
  2. Click Add Files… or Add Folder… to load your video files (.mp4, .mov)
  3. Adjust Noise Level and Minimum Duration if needed (defaults work well for most footage)
  4. Click Start Processing
  5. When complete, select your export format (Final Cut Pro / DaVinci Resolve) and handling mode, then click Generate

Frequently Asked Questions

What video formats are supported?

Any format supported by macOS AVFoundation — including H.264, H.265/HEVC, ProRes, and most camera-native formats (.mp4, .mov, .m4v).

How does the detection work?

FDUtility uses a two-phase algorithm. Phase 1 scans timestamp gaps in VFR (Variable Frame Rate) content without decoding — this is how it achieves 1,600× realtime speed. Phase 2 uses pixel-level comparison (Mean Absolute Difference via Apple's Accelerate/vDSP) for CFR content.

What do the detection parameters mean?

Noise Level (default: -60 dB): How similar frames must be to count as "frozen". Lower values (more negative) = more sensitive. -60 dB means frames must differ by less than 0.1% in average luma.

Minimum Duration (default: 1.0s): The shortest freeze to report. Freezes shorter than this threshold are ignored.

What FCPXML export modes are available?

  • Gaps — Clean footage becomes clips; frozen segments become gaps (empty space)
  • Frozen Clips — Both frozen and clean segments appear as clips for in-context review
  • Non-Freezes Only — Frozen segments are removed entirely

Can I import the FCPXML into DaVinci Resolve?

Yes. FDUtility exports FCPXML v1.8 for DaVinci Resolve and FCPXML v1.10 for Final Cut Pro. Select either or both formats before clicking Generate.

Known Issues

Final Cut Pro 12.0 crashes on FCPXML import Apple Bug

FCP 12.0 on macOS Tahoe 26.x has a confirmed bug that crashes on any FCPXML import containing <asset-clip> elements — including FCP's own exported XML. This is an Apple regression unrelated to FDUtility. Use FCP 11.x until Apple releases a patch.

System Requirements

  • No external dependencies required
