Privacy Policy

FDUtility — Frozen-Frame Detection

Overview

FDUtility is a fully offline macOS application for detecting frozen frames in video files. Your privacy is straightforward: we collect nothing.

Data Collection

FDUtility does not collect, transmit, store, or share any personal data, usage data, analytics, crash reports, or telemetry of any kind.

Network Access

FDUtility makes zero network requests. The application operates entirely offline. Your video files and detection results never leave your machine.

File Access

FDUtility accesses only the video files you explicitly select through the macOS file picker or folder picker. It reads video frames for analysis and writes FCPXML export files to the same directory as your source media. No other files or directories are accessed.

Third-Party Services

FDUtility uses no third-party SDKs, analytics frameworks, advertising networks, or cloud services. It is built entirely with Apple's native frameworks:

  • AVFoundation (video decoding)
  • Accelerate / vDSP (frame comparison)
  • SwiftUI and AppKit (user interface)

Children's Privacy

FDUtility does not collect data from any user, including children under 13.

Changes to This Policy

If this policy changes, the updated version will be posted at this URL. Since FDUtility collects no data, material changes are unlikely.
